# Regulatory Submission Workflow Tracker

## Project Overview
This project simulates an end-to-end regulatory submission workflow inspired by regulatory information management platforms used in the life sciences industry.

It demonstrates how regulatory teams can track submission planning, content plans, document lifecycle status, publishing readiness, agency queries, and archival using synthetic data.

## Disclaimer
This is an educational portfolio project. It is not affiliated with, endorsed by, or copied from Veeva Systems. All data is synthetic and created for demonstration purposes.

## Business Problem
Regulatory teams manage multiple submissions across regions, health authorities, document owners, and timelines. Manual tracking can create delays, missing documents, version control issues, and poor visibility.

## Objective
To create a simple regulatory submission workflow tracker that helps monitor:
- Submission status
- Required documents
- Document review and approval
- Publishing readiness
- Agency queries
- Archive status
